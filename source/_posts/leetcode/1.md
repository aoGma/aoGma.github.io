## [两数之和](https://leetcode.cn/problems/two-sum/description/)

通过设置字典，保存之前的结果，如果在之前的结果中找到，返回即可

```js
/**
 * @param {number[]} nums
 * @param {number} target
 * @return {number[]}
 */
var twoSum = function (nums, target) {
 const map = new Map();
 for (let i = 0; i < nums.length; i++) {
  const _target = target - nums[i];
  if (map.has(_target)) {
   return [map.get(_target), i];
  } else {
   map.set(nums[i], i);
  }
 }
};
```
