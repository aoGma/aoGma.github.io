## [最长公共前缀](https://leetcode.cn/problems/longest-common-prefix/description/)

循环判断字符串的每一位

```js
/**
 * @param {string[]} strs
 * @return {string}
 */
var longestCommonPrefix = function (strs) {
 let i = 0;
 while (strs.every((o) => i < o.length && o[i] === strs[0][i])) {
  i++;
 }
 if (i === 0) return "";
 return strs[0].slice(0, i);
};
```
