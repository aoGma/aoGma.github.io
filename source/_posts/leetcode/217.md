## [存在重复元素](https://leetcode.cn/problems/contains-duplicate/description/)

使用set判断

```js
/**
 * @param {number[]} nums
 * @return {boolean}
 */
var containsDuplicate = function (nums) {
 const set = new Set();
 for (let i = 0; i < nums.length; i++) {
  if (!set.has(nums[i])) {
   set.add(nums[i]);
  } else {
   return true;
  }
 }
 return false;
};
```
