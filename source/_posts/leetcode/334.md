## [递增的三元子序列](https://leetcode.cn/problems/increasing-triplet-subsequence/description/)

保存两个最小的数，只要找到第三个数就得到了答案

```js
function increasingTriplet(nums) {
  let first = Infinity;
  let second = Infinity;
  for (const n of nums) {
    if (n <= first) {
      first = n;
    } else if (n <= second) {
      second = n;
    } else {
      return true;
    }
  }
  return false;
}
```
