## [爬楼梯](https://leetcode.cn/problems/climbing-stairs/description/)

动态规划
因为最后一步只能选择1、2
公式：dp[n] = dp[n-1] + dp[n-2]

```js
/**
 * @param {number} n
 * @return {number}
 */
var climbStairs = function (n) {
 if (n === 1) return 1;
 if (n === 2) return 2;
 const dp = [1, 2];
 for (let i = 2; i < n; i++) {
  dp.push(dp[i - 1] + dp[i - 2]);
 }
 return dp.pop();
};
```

优化空间

```js
/**
 * @param {number} n
 * @return {number}
 */
var climbStairs = function (n) {
 if (n === 1) return 1;
 if (n === 2) return 2;
 const dp = [1, 2];
 for (let i = 2; i < n; i++) {
  // dp.push(dp[i - 1] + dp[i - 2]);
  const next = dp[0] + dp[1];
  dp[0] = dp[1]
  dp[1] = next;
 }
 return dp.pop();
};
```
